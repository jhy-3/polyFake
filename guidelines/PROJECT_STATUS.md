## 🏗️ 架构演进

### v0.1 - 概念阶段（overview.md）
- **定位**: 黑客松项目构思
- **技术栈**: 未定
- **存储**: PostgreSQL（建议）
- **前端**: Streamlit/Next.js（建议）

### v1.0 - Streamlit 版本
- **定位**: 快速原型验证
- **技术栈**: Python + Streamlit
- **文件**: `polysleuth/dashboard_pro.py`, `dashboard_real.py`
- **存储**: 内存处理
- **状态**: 已完成但不再主要维护

### v2.0 - 生产架构（当前）
- **定位**: 生产级应用
- **技术栈**: FastAPI + 原生 JS + WebSocket
- **文件**: `backend/`, `frontend/`
- **存储**: SQLite + 内存混合架构
- **状态**: ✅ 主要开发版本
